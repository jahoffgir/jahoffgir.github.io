Avalon - Introduction

Avalon is a fanmade web-application game, implemented in HTML, CSS, and JavaScript, that was inspired by The Resistance: Avalon board game. Players can play the game in a group. Currently, the game can be played on one device. Rules of the game and open-source code can be found here, respectively: rules and code.

How to

When initially starting the game, users will be instructed to type how many people are playing. Afterwards, everyone types their alias and once thats done, each person checks their character(WITHOUT revealing their character), and information about the players. Next, the 'King' picks the players that will go on the mission. For each mission, other players need to accept or reject the people that are going to the mission. This will repeat until good or evil wins.

Backstory

When my friends and I attended hackathon, Hack Upstate, we did not know what we wanted to work on, but we knew that we wanted to make an awesome web-application game. Upon researching, we set our mind to making the fanmade Avalon game. When researching, we could not find a version that was a web-application, free, and user friendly. Our goal was to make the game user friendly, open-source, and robust.
